# Phase 06 Fix Report

## 1 & 2. Export / Import Project and Auto-Naming
**Files changed**: `src/utils/fileSystem.js`, `src/utils/index.js`, `src/components/TopBar.jsx`
Changed the methods `openFile`/`saveFile` to `importProject`/`exportProject`, updated the UI buttons in `TopBar.jsx` to display "Import" and "Export", and added an automatic filename fallback so the browser proposes a snake_case version of the current project title for download.

## 3 & 4. Side Effects Flex Wrap and Delete Icon
**Files changed**: `src/components/NodeInspector.jsx`, `src/components/EdgeInspector.jsx`
Replaced the glitchy "DEL" and "ÁE" text in the Inspector side effect lists with a clear `🗑️` universal trash emoji, while injecting `flexWrap: 'wrap'` into the container style to gracefully wrap long flag names without overflowing the actions.

## 5. Edges Arrow Indicator
**Files changed**: `src/components/GraphCanvas.jsx`
Imported `MarkerType` from `@xyflow/react` and dynamically applied `markerEnd: { type: MarkerType.ArrowClosed, color: '#b1b6be' }` to all mapped conditional edges to clarify path directionality.

## 6. Readable JSON Metadata Dates
**Files changed**: `src/store/graphStore.js`
Transformed the Unix timestamps in `exportGraph` output via a custom `formatTs` local closure to export dates natively as `DD-MM-YYYY` explicitly when packaging the `meta` bundle.

## 7. Weird Start Node Indicator Typo
**Files changed**: `src/components/NodeInspector.jsx`
Repaired a corrupted text-encoding string glitch by correcting `Start Node ✁E` to `Start Node ✓`.
